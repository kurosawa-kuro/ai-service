## 役割分担（現契約での使い分け）

- **設計・レビュー・重い実装** → Claude Code（Claude Max 5x）。
- **repo 実装・docs 同期・CLI エージェント** → Codex（ChatGPT Pro）。
- **IDE 内の補完・軽い修正** → GitHub Copilot Pro。
- **AI IDE 体験の試用** → Cursor Hobby（無料枠）。有料化判断は保留。

## 見直しの観点（コスト最適化）

- **Claude Max 5x ($100)**: Claude Code のメイン。使用量が 5時間枠/週次上限に頻繁に当たるなら Max 20x ($200) へ、余っているなら Pro ($20) へ下げる余地。→ 実使用量を観察して判断。
- **ChatGPT Pro ($100)**: Codex メイン。Codex 中心で quota が余るなら Plus ($20) で足りる可能性。逆に quota 逼迫なら Pro $200。
- **重複の点検**: Claude Code / Codex / Copilot で**コーディング支援が三重**。実作業でどれをどれだけ使うかを [evaluations/](evaluations/) で計測し、被りが大きければ 1つを格下げ/解約する候補。
- **合計 $210/月（≈¥32,550）** は個人としては小さくない。用途量の実測（`evaluations/`）で「定額サブスク vs BYO-key 従量」の損益分岐を出すと最適化しやすい。
