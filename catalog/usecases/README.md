# 用途別逆引き

サービス名ではなく「何をしたいか」から候補を引くためのページ。

各ページは次の構造を持つ。

- 第一候補
- 低コスト補助
- ローカル検証
- 判断ルール
- 破綻条件
- 関連評価ログ

## 初期用途

| 用途 | ファイル | 見る場面 |
|---|---|---|
| コーディング | `coding.md` | 実装、レビュー、テスト、docs更新 |
| 調査 | `research.md` | 最新情報、技術調査、出典確認 |
| ローカル / 秘匿 | `local-secret.md` | 機密寄り、低コスト大量処理 |
| ハーネス / ルーティング | `harness-routing.md` | 自作AIルーター、API比較、MCP |
| 講座 / 発信 | `course-content.md` | 講座、YouTube、SNS素材 |
| プレゼン / 資料 | `presentation.md` | 営業資料、PM資料、提案書 |
| PM / 業務自動化 | `pm-automation.md` | 顧問、会議、タスク、業務設計 |
| トレード材料整理 | `trading-research.md` | 市場材料、世界情勢、ニュース監視 |
