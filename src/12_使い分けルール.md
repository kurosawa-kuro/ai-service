# 使い分けルール

用途から AI サービス / 技術を選ぶための使い分け判断の正本。

## 基本方針

- 能力カテゴリ、用途、ハーネス適性、評価ログを根拠に選ぶ。
- 印象評価だけでルーティングしない。
- 機密に近い内容はローカル実行または契約上許可された環境に限定する。
- 「契約を増やす / 減らす」ではなく、増え続ける AI を比較可能な能力資産として扱う。
- 価格だけで判断しない。API / CLI / MCP / AGENTS.md / ローカル可否 / ログの残しやすさを必ず見る。
