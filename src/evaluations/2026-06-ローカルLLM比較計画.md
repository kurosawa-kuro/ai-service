# 2026-06 local LLM comparison plan

## タスク

ローカルLLM環境を、秘匿寄り処理・低コスト大量処理・講座デモの観点で比較する。

## 入力

- 日本語の長文メモ
- 技術ドキュメントの抜粋
- 議事録風テキスト
- 分類・要約・抽出タスク

## 使用サービス / 技術

- Ollama
- LM Studio
- LocalAI
- Open WebUI
- Qwen系モデル
- DeepSeek系モデル

## 比較タスク

| タスク | 見ること |
|---|---|
| 日本語要約 | 読みやすさ、抜け漏れ |
| 分類 | 一貫性、速度 |
| 固有表現抽出 | 正確性 |
| ローカルRAG接続 | Qdrant / Chroma / pgvector との相性 |
| 講座デモ | 見せやすさ、再現性 |

## 評価軸

| 軸 | A | B | C |
|---|---|---|---|
| 秘匿性 | 完全ローカルで完結 | 一部外部依存 | 外部送信あり |
| 品質 | 実務補助に使える | 下処理なら使える | 手戻りが大きい |
| 速度 | 反復可能 | 待てば使える | 遅い |
| 操作性 | 日常運用できる | デモ向き | 面倒 |
| 自動化 | CLI / APIで回せる | 半自動 | GUIのみ |

未実施。

## 再利用判断

- 秘匿寄り素材はまずローカルで下処理する。

- 講座デモ用途では GUI の見せやすさも評価する。
