# モデル台帳

**プロバイダ × モデル階層**の参照データ。

`21_サービス/` が「ツール（Claude Code、Cursor 等）」を扱うのに対し、
ここは「**モデルそのもの**（Fable 5、GPT-5.6 Sol 等）」を扱う。
契約判断の主語がサービスからモデルへ移ったため分離した。

| ファイル | 役割 |
|---|---|
| [モデル階層.md](モデル階層.md) | 役割別にどのモデル階層を当てるか（仮説・実測前） |
| [プラン比較.md](プラン比較.md) | コーディング系サービスの料金プラン横比較 |

## サービスとの違い

| | 21_サービス/ | 20_モデル/ |
|---|---|---|
| 単位 | ツール・サービス | モデル・プラン階層 |
| 例 | Claude Code、Cursor、Perplexity | Fable 5、GPT-5.6 Sol、Max 5x |
| 主な使い道 | 用途別の選定 | 契約判断・コスト判断 |
