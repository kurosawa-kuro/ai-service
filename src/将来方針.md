# 将来方針（AI サービス契約の目標形）

> 契約の**将来の目標形**を管理するドキュメント。現在実際に払っているものは [現在の契約.md](現在の契約.md)（このファイルとは疎結合で管理し、現契約の変更でここは書き換えない。方針が変わったときだけ更新する）。
> **記録日: 2026-07-10**（本人申告）。

## 目標形（時期未定）

| 枠 | 目標プラン | 月額目安 | 位置づけ |
|---|---|---:|---|
| OpenAI | ミドルレベル（ChatGPT Plus $20 相当） | 約¥3,000 | 高性能モデルを量限定で利用 |
| Anthropic | ミドルレベル（Claude Pro $20 相当） | 約¥3,000 | 高性能モデルを量限定で利用 |
| コーディング | **Cursor へ移行** | Pro $20〜 | Cursor が躍進しそうなため。プライベートアプリも一巡し、そこそこの性能で充分の見込み |

## 考え方

- 二大フロンティア（OpenAI / Anthropic）は「上位モデルに少量アクセスできるミドル階層」を両方保持する。
- コーディングの主力は Codex / Claude Code から Cursor 側へ寄せる。
- コスト: 現行 $310/月（2026-07 時点）→ 目標 $60〜/月 の大幅圧縮。

## 移行の前提確認（実測してから切り替える）

1. Plus / Pro のミドル階層で上位モデル（GPT-5.6 Sol / Fable 5 等）に実際どこまで触れるか。
2. Cursor の使用量クレジット（Pro $20 プール）が実作業量に足りるか。
3. Codex / Claude Code から Cursor へ移して困る作業（CLI エージェント・repo 横断）の代替。

→ [evaluations/](evaluations/) で実測してから段階移行。最初の判断点は 2026-08-07（Claude Max 20x の一時アップグレード期限）。
