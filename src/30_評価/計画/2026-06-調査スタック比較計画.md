# 2026-06 調査スタック比較計画

## タスク

AIサービス / 技術の最新情報を調べる用途で、検索・調査系ツールを比較する。

## 入力

- 調査テーマ: AIコーディングエージェントの CLI / IDE / MCP 対応
- 必須確認: 公式 docs、料金、API、CLI、MCP、利用規約
- 出力形式: サービス台帳へ転記できる表

## 使用サービス / 技術

- Perplexity
- NotebookLM
- Tavily
- Exa
- Firecrawl
- 通常検索
- Claude / ChatGPT による整理

## 比較タスク

| タスク | 見ること |
|---|---|
| 公式ページの発見 | 一次情報に到達できるか |
| 料金・無料枠確認 | 古い情報を混ぜないか |
| API / CLI / MCP確認 | ハーネス適性の判断に使えるか |
| 複数サービス比較 | 出典つきで表にできるか |
| 台帳転記 | `services/*.md` の項目へ落とせるか |

## 評価軸

| 軸 | A | B | C |
|---|---|---|---|
| 出典品質 | 公式・一次情報中心 | 二次情報混在 | 出典弱い |
| 鮮度 | 日付・現行情報を確認 | 一部不明 | 古い |
| 抽出精度 | 台帳項目にそのまま使える | 手直し必要 | 抜け漏れ多い |
| 自動化適性 | API / 構造化出力あり | 半自動 | 手作業中心 |
| 検証容易性 | 出典へ戻れる | 戻りにくい | 戻れない |
